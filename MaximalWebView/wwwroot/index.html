Recently, I've been interested in the intersection of native apps and web UI. It's a good time for that:

WebView2

Windows 11 ships with WebView2, a modern "webview" control based on Edge/Chromium. It's Windows-only for now, but it will be coming to macOS and Linux soon.

WebView2 is good Windows software. It exposes a powerful but straightforward API over COM for any Windows application to use, all the way back to Windows 7. You can tell that Microsoft wants to make it easy to adopt, it's a breath of fresh air after the last decade.

For many applications, WebView2 can deliver a better user experience than native UI frameworks - today. Think of it as Electron with fewer downsides.

.NET

The .NET team has been killing it. We're now able to ship small, self-contained .NET executables with great startup time and steady-state performance on any platform. But cross-platform UI has remained elusive, until now.
